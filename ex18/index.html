Hexbin

D3 map visual
Bubble Chart
I like how the bubbles on the map is showing the population density. I prepared a bubble chart for my tableau exercise and it is an impressive tool to visualize population density. It is a nice visual respresentation showing which counties have more people and where the population is more concentrated in the U.S. The code can be altered based on width, height and radius of the map and circles. The colors of the circles can also be modified.
